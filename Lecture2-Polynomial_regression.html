Based on this data, our objective is to build a regression model of the form \[y_i = f(x_i) + e_i \quad ; \quad 1 \leq i \leq n\] where \(y\) denotes the weight of the rats and \(x\) the number of week. We will restrict ourselves to polynomial regression, by considering functions of the form \[ \begin{aligned} f(x) &= f(x ; \beta_0, \beta_1, \beta_2, \ldots, \beta_d) \\ &= \beta_0 + \beta_1 x + \beta_2 x^2 + \ldots + \beta_d x^d \end{aligned} \]

[Polynomial regression model]

The polynomial regression model \[y_i=\beta_0+\beta_1x_i+\beta_2x_i^2+\cdots+\beta_dx_i^d+e_i\] can be written in a matrix form \[Y=X\beta+\boldsymbol{e}\] where \(Y=(y_1,\cdots,y_n)^\top\), \(\boldsymbol{e}=(e_1,\cdots,e_n)^\top\) and \[X=\begin{pmatrix} 1&x_1&x_1^2&\cdots&x_1^d\\ 1&x_2&x_2^2&\cdots&x_2^d\\ \vdots&\vdots&\ddots& &\vdots\\ 1&x_n&x_n^2&\cdots&x_n^d\\ \end{pmatrix}\quad\text{and}\quad \beta=\begin{pmatrix} \beta_0\\ \beta_1\\ \vdots\\ \beta_d\\ \end{pmatrix}\]

Confidence interval and prediction intervals

‚ö†Ô∏è Note that

  • Confidence interval is a interval for \(f(x)\)
  • Prediction interval is a interval for \(y = f (x) + e\).

The second one is larger than the first one! This prediction interval takes into account both the uncertainty on the predicted distance \(f(x)\) and the variability of the measure, represented in the model by the residual error \(e\)

4. Fitting an orthogonal polynomial model (use default: raw=FALSE)

Let \(\boldsymbol{x}=(x_{1},\cdots,x_{n})^\top\in\mathbb{R}^n\). The columns of \(X=[\mathbf{1}_n,\boldsymbol{x},\boldsymbol{x}^2, \ldots,\boldsymbol{x}^d]\) are highly correlated. The Gram matrix is such \[ G=\frac{1}{n}[X^\top X]_{k\ell}=\frac{1}{n}\sum_{i=1}^nx_i^{k+\ell-2} \]

  • cov2cor() transforms this covariance matrix into a correlation matrix, where each off-diagonal element represents the correlation between pairs of variables.

4.1. The Gram-Schmidt procedure

Let us explore how to use a sequence of orthogonal polynomials to obtain uncorrelated estimates. Let \(\boldsymbol{x}=(x_{1},\cdots,x_{n})^\top\in\mathbb{R}^n\). The columns of \(X=[\mathbb{1}_n,\boldsymbol{x},\boldsymbol{x}^2, \ldots,\boldsymbol{x}^d]\) are highly correlated, as can be seen from the relationship: \[ [X^\top X]_{k\ell}=\sum_{i=1}^nx_i^{k+\ell-2} \]

By employing orthogonal polynomials, we aim to reduce this multicollinearity and achieve more stable estimates. Orthogonal polynomials can be obtained by applying the following Gram-Schmidt orthogonalization process to the basis \(\{\mathbb{1}_n,\boldsymbol{x},\boldsymbol{x}^2, \ldots,\boldsymbol{x}^d\}\)

\[\begin{align*} p_0(\boldsymbol{x})&=1\\ p_1(\boldsymbol{x})&=\boldsymbol{x}-\frac{\langle \boldsymbol{x},p_0\rangle}{\langle p_0,p_0\rangle}p_0(\boldsymbol{x})\\ p_2(\boldsymbol{x})&=\boldsymbol{x}^2-\frac{\langle \boldsymbol{x}^2,p_1\rangle}{\langle p_1,p_1\rangle}p_1(\boldsymbol{x})-\frac{\langle \boldsymbol{x}^2,p_0\rangle}{\langle p_0,p_0\rangle}p_0(\boldsymbol{x})\\ &\vdots\\ p_d(\boldsymbol{x})&=\boldsymbol{x}^d-\frac{\langle \boldsymbol{x}^d,p_{d-1}\rangle}{\langle p_{d-1},p_{d-1}\rangle}p_{d-1}(\boldsymbol{x})-\cdots-\frac{\langle \boldsymbol{x}^d,p_0\rangle}{\langle p_0,p_0\rangle}p_0(\boldsymbol{x}) \end{align*}\]

Then \(\widetilde X=[p_0(x),p1(x),\cdots,p_d(x)]\) is the new design matrix such that \[(\widetilde X^\top \widetilde X)^{-1}=\begin{pmatrix} n&0&0&\cdots&0\\ 0&1&0&\cdots&0\\ \vdots&\vdots&\ddots& &\vdots\\ 0&0&0&\cdots&1\\ \end{pmatrix}\in\mathbb{R}^{p\times p} \]

6. Data transformation

When fitting a linear regression model one assumes that there is a linear relationship between the response variable and each of the explanatory variables. However, in many situations there may instead be a non-linear relationship between the variables. This can sometimes be remedied by applying a suitable transformation to some (or all) of the variables, such as power transformations or logarithms.

In addition, transformations can be used to correct violations of model assumptions such as constant error variance and normality.

Applying suitable transformations to the original data prior to performing regression may be sufficient to make linear regression models appropriate for the transformed data.
